## Keybind

* [sxhkdrc](sxhkdrc)

| Keybind           | Action      | Command                         |
| ----------------- | ----------- | ------------------------------- |
| `Alt + Shift + d` | Launch Rofi | `rofi -show drun -show-icons`   |
| `Alt + Shift + w` | Launch Rofi | `rofi -show window -show-icons` |
| `Alt + Shift + r` | Launch Rofi | `rofi -show run`                |


## Howto

### Start sxhkd

Run

``` sh
sxhkd -c sxhkdrc
```
